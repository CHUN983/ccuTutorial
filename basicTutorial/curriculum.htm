HTML 基礎課程 - 網頁設計入門

課程目標

本課程將介紹如何建立一個基本的網頁結構，並涵蓋 HTML、CSS 和 JavaScript 的基礎知識。學生將學習如何設計網頁，並學會如何使用 CSS 製作美觀的版面布局。

課程大綱

  • HTML 基礎結構 - 介紹 HTML 標籤和網頁結構。
  • HTML 元素 - 了解網頁元素，如標題、段落、圖片等。
  • CSS 樣式設計 - 學習如何使用 CSS 設計頁面佈局。
  • JavaScript 基礎 - 介紹如何使用 JavaScript 進行網頁交互。

1. HTML 基礎結構

HTML 是用來設計網頁結構的標記語言。它由一系列標籤組成，通常包括 <html>，<head> 和 <body>。

<html>
    <head>
        <meta charset="UTF-8">
        <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css">
    </head>
    <body>
        <header>
            <img src="01.png" height="150px" width="100%">
        </header>
        <main>
            <iframe name="contentFrame" src="main.html" width="100%" height="100%" style="border:none;"></iframe>
        </main>
    </body>
</html>

2. HTML 元素

HTML 元素用來描述網頁中的各種內容，如標題、圖片、連結等。以下是一些常用的 HTML 元素：

<footer>
    <a href="https://www.facebook.com/SUMMER.CCUCSIECamp/?locale=zh_TW">
        <i class="fab fa-facebook-f"></i>
    </a>
    <a href="https://www.instagram.com/ccucsie/">
        <i class="fab fa-instagram"></i>
    </a>
    <a href="https://www.threads.net/@ccucsie">
        <i class="fab fa-threads"></i>
    </a>
    <p>© 2025 我的網站. 版權所有.</p>
</footer>

3. CSS 樣式設計

CSS 用來控制網頁的外觀，像是文字顏色、背景顏色、元素排列等。以下程式碼展示了如何使用 CSS 設定頁面的基本樣式：

* {
    margin: 0;
}

header {
    height: 150px;
    position: sticky;
    top: 0;
}

#menu {
    background-color: gray;
    position: sticky;
    top: 150px;
}

#menu ul {
    padding: 0;
    display: table;
    width: 100%;
    list-style-type: none;
}

#menu li {
    display: table-cell;
    background-color: white;
    font-size: 24px;
    border: 1px black dashed;
}

#menu li a {
    display: block;
    text-align: center;
    text-decoration: none;
}

#menu li a:hover {
    color: white;
    background-color: rgb(69, 59, 56);
}
